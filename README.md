 🍱 Ghar Se Thali

Ghar Sa Thali is a smart food delivery and subscription platform that connects customers with home chefs (aunties) who prepare fresh, hygienic, and affordable meals.

🚀 Features

- User Login & Signup (OTP / Email)
- Weekly & Monthly Subscription Plans
- Daily Menu Display
- Online Payment System (UPI, Card, Wallet)
- Order Tracking System
- Wallet & Transaction History
- Feedback & Rating System
- Admin Dashboard

 👨‍💻 Tech Stack

Frontend:
- React.js / HTML / CSS / Tailwind

Backend:
- Node.js / Express.js

Database:
- MongoDB

Payment Integration:
- Razorpay / Stripe

 📱 Use Case

This app is designed for:
- Students
- Office workers
- Bachelors
- Families

 💡 Project Objective

To provide healthy, home-made food at affordable prices with a flexible subscription model.

🔮 Future Scope

- AI-based food recommendations
- Multi-city expansion
- Vendor (Home Chef) onboarding system
- Mobile app launch (Android & iOS)
